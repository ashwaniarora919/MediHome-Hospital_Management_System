# MediHome - Hospital Management System

A full-stack Hospital Management System developed using Java, JSP, Servlets, JDBC, MySQL, Bootstrap, HTML, CSS, and Tomcat.

## Features

### Admin Module

* Admin Login
* Add Doctor
* View Doctors
* Edit Doctor
* View Users
* View Appointments
* Dashboard Statistics

### Doctor Module

* Doctor Login
* View Assigned Patients
* Patient Appointment Status

### User Module

* User Registration
* User Login
* Book Appointment
* View Appointment History

## Technologies Used

* Java
* JSP
* Servlet
* JDBC
* MySQL
* Bootstrap
* HTML/CSS
* Apache Tomcat

## Database

MySQL Database used for storing:

* Users
* Doctors
* Appointments
